Venga translator module
-----------------------

Integrates the Vengacorp translation services with the TMGMT module.
https://drupal.org/project/tmgmt


Installation
------------

 * Copy the whole module directory to your modules directory
   (e.g. DRUPAL_ROOT/sites/all/modules) and activate it "admin/modules".
 * The module provides a new translation source. Go to
   "admin/config/regional/tmgmt_translator/manage/venga_translator" to adapt
   the XTRF credentials & Workflow options.
 * Start using by creating translation jobs.
